# Redirects

`public/_redirects` contains an initial, conservative redirect map for routes
from the prior personal site.

## Must Verify Before Launch

- `/blog.html` should land at `/writing/`.
- `/commitments.html` should land at a final principles or about section.
- `/pulpit-progress.html` should leave this career site and go to the Pulpit
  product's public progress route.
- `/posts/template.html` should not remain publicly indexed.

Do not copy old source notes or generated tracker files into this repository.
