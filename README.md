# noahairmet.com

Clean-room foundation for Noah Airmet's professional website.

## Stack

- Astro static output
- Strict TypeScript
- Astro content collections for work and writing
- Hand-authored CSS in small layer files
- Cloudflare Workers Static Assets serving `dist`

## Commands

```bash
npm run dev
npm run build
npm run check
npm test
npm run preview
```

`npm run preview` expects `dist` to exist. Build first.

## Current State

This repository intentionally contains a neutral shell, placeholder routes, and
mechanical deployment/test scaffolding. Final visual direction, homepage
composition, case-study writing, and launch redirects are owned by the lead
site session.
